Configure your BitcoinPriceLED

When Static Color is enabled BitcoinPriceLED will be lit in a static color.

Please enter the HEX Value of the desired color (e.g. #ffffff)



When Nightmode is enabled BitcoinPriceLED will be dimmed between the selected hours.





Interval sets the time (in seconds) that BitcoinPriceLED will wait before fetisching a new price.
